HireGround: Product Requirements Document (PRD)

📋 Executive Summary

HireGround delivers context-aware resume screening that eliminates institutional bias by normalizing candidate scores based on real-world opportunity gaps. Unlike uniform models favoring metro candidates, HireGround provides equitable shortlisting while maintaining hiring quality.

Core Value: Transform raw skills evaluation into opportunity-adjusted fairness.
🎯 Problem Statement

Traditional resume screeners apply uniform standards, systematically disadvantaging candidates from low-opportunity backgrounds:

Metro Tier-1 Candidate: Elite internships → Score: 85 → Shortlist
Tier-3 Rural Candidate: Equivalent skills, limited exposure → Score: 65 → Rejected

Root Cause: One-size-fits-all evaluation ignores structural disparities in access, education quality, and job markets.
👥 Target Users & Personas
Persona	Role	Key Need
HR Recruiter	Corporate/Startups	Diverse shortlists without quality compromise
Startup CTO	Tech hiring	Bias audits + compliance safety
Hackathon Judge	Academic/Industry	Quantified fairness metrics + working demo
🎮 Core Features
1. Resume Parsing

Input: PDF/DOCX resume
Output: Structured candidate profile

Extracts: skills, college, location, CGPA, projects, internships, certifications

2. Dual Scoring Pipeline

Baseline Score: Skills + Experience + Education = X/100
Context Adjustment: Opportunity-based credits = +Y points
Final Score: min(100, Baseline + Adjustment)

3. Side-by-Side Comparison

Batch analytics showing uniform vs. equitable outcomes
4. Fairness Passport

Per-candidate explainability:

"Baseline: 72 → Final: 85 (+13 credit)"
Breakdown: "Tier-3 college (+8), Rural district (+5)"

5. Batch Processing

Upload 100+ resumes → Group fairness metrics + ranking shifts

CandidateScore

python
class CandidateScore(BaseModel):
    candidate_id: str
    baseline_score: float                # Uniform model: 72.5
    context_adjustment: int              # Opportunity credits: +13
    final_score: float                   # Equitable score: 85.5
    rank_uniform: Optional[int]          # Batch rank without context
    rank_final: Optional[int]            # Batch rank with context
    fairness_passport: FairnessPassport

FairnessPassport

python
class FairnessPassport(BaseModel):
    baseline_score: float
    total_adjustment: int
    adjustment_breakdown: List[str]      # ["Tier-3 college (+8)", "Rural district (+5)"]
    rank_shift: Optional[str]            # "15 → 3"
    ceos_group_score: float              # Contextual Equity Score: 87.2
